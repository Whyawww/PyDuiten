# PyDuiten

Aplikasi pencatat keuangan modern, cepat, dan aman. Dirancang untuk membantu pengguna memantau arus kas, menganalisis pengeluaran bulanan, dan mendapatkan wawasan (insight) cerdas secara real-time.

## Fitur Utama

* **Dashboard Real-time:** Pantau pemasukan dan pengeluaran dengan antarmuka yang minimalis dan responsif.
* **Smart Insight Engine:** Analisis Month-on-Month (MoM) otomatis yang memberikan evaluasi tren pengeluaran langsung di sisi klien (Deterministic Client-Side Logic), memastikan tidak ada data yang terbuang sia-sia.
* **Ekspor Laporan PDF:** Unduh ringkasan keuangan dalam format PDF yang di-generate secara 100% type-safe.
* **Keamanan Berlapis:** Sistem autentikasi terintegrasi untuk melindungi data pribadi pengguna.
* **Teruji Menyeluruh:** Kode terproteksi dengan Unit Testing (Vitest) dan End-to-End Testing (Playwright) untuk menjamin stabilitas fungsionalitas di lingkungan produksi.
